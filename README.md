# Dev
## requirement
- Node.js
- yarn

## init

clone and

```fish
$ yarn install
```

## scripts
See `package.json`

### release build

```fish
$ yarn release
```

### build with distributions (for dev)

```fish
$ yarn build
```

bundled with develop mode js files will be generated in `/dist` 

### lint

```fish
$ yarn lint
```

eslint & prettier
